### Cảm ơn bạn vì đã ghé ngang. :v:

Mình là Lợi.

Mình mới bắt đầu học code bán nghiêm túc trong vài tháng gần đây ^^.

Mình có mặt ở đây tất nhiên là mình đã phần nào vượt qua được trải nghiệm tồi tệ đó ^^.
